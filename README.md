DNS Reconnaissance Tool
=======================

This project is an improvement over darkoperator's original DNSRecon tool.


Requirements
------------

* Python 2.7


Usage
-----


```
git clone
virtualenv -p python2 dnsrecon
cd dnsrecon
. bin/activate
pip install -r requirements.txt
./dnsrecon.py
```
